# Getting Started

## Installation

```bash
pip install model-canary
```

## Quick Start

1. Initialize a project:
   ```bash
   model-canary init
   ```

2. Set your API keys:
   ```bash
   export OPENAI_API_KEY="sk-..."
   export ANTHROPIC_API_KEY="sk-ant-..."
   ```

3. Run canary tests:
   ```bash
   model-canary run
   ```

4. View the dashboard:
   ```bash
   model-canary dashboard
   ```
